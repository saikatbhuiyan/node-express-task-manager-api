## Backend API for task manager app

### Features

- Create, Update, Delete and Get User
- Login Logout User
- Send Email
- Image Upload
- Create, Update, Delete and Get Task
- Test Features

### Clone project

```sh
git clone ...
cd task-manager
```

### Install dependencies

```sh
npm install
```

### Run server

```sh
npm run start
```

### Run dev server

```sh
npm run dev
```

### Run test

```sh
npm run test
or
node jest
```
